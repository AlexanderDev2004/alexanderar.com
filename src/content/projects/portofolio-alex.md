## Project Summary

This personal portfolio is a total overhaul of my old website, built using Astro, Svelte, Tailwind CSS, and Bun.
I designed it to be fast, clean, and easy to navigate, so visitors can quickly get a feel for my projects, skills, and journey as a software developer.

## The Problem
So, my old personal website actually worked. It loaded. My name was on it. But every time I sent the link to someone, there was this little voice in the back of my head saying, "this doesn't represent my style at all." Eventually, I just thought, "screw it, I might as well just make a new one." So, I built this. And you can see the old version here: https://alexanderar.vercel.app/

## The Solution
I rebuilt my personal website from scratch using Astro, Svelte, Tailwind CSS, and Bun, focusing on creating something that truly felt like me while still looking professional. The goal wasn't to chase trends, but to make it recognizable and authentic.

## Key Features

### Content Management
- Structured content with Astro Content Collections for projects, blogs, and bookmarks.
- Markdown-based writing flow for easier updates without touching component logic.
- Consistent metadata (title, year, links, tags, and technologies) to keep entries organized.

### Design & UX
- Responsive layout designed for both desktop and mobile reading.
- Clean visual hierarchy to highlight key sections, projects, and personal story.
- Clear interaction cues, including styled links and readable typography.

### Technical Features
- Static site generation for fast load times and reliable performance.
- Reusable Svelte components to keep UI patterns consistent across pages.
- SEO-friendly metadata setup (title, description, canonical, and Open Graph tags).

### Technical Stack
- Astro for routing, content rendering, and static output.
- Svelte for interactive UI components.
- Tailwind CSS for utility-first styling and rapid iteration.
- Bun for dependency management and development workflow.
- TypeScript for stronger typing and safer refactoring.

### Design Philosophy
- Prioritize clarity before complexity.
- Keep the interface personal, intentional, and easy to explore.
- Balance modern aesthetics with performance and maintainability.

### Challenges & Learnings
- Learned to balance visual polish with performance and readability.
- Improved my approach to structuring scalable, content-driven pages.
- Gained practical experience combining Astro and Svelte in one cohesive project.
